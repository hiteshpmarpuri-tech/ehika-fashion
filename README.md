# Ehika Fashion

This repository contains a starter Next.js + Tailwind site for "Ehika Fashion" — a luxurious shopping site prototype.

Quick start:

1. Copy .env.local and set ADMIN_TOKEN if you want a real token.
2. npm install
3. npm run dev
4. Open http://localhost:3000

Admin:
- Visit /admin and enter the ADMIN_TOKEN (the default in .env.local is a placeholder). The admin UI posts to protected API routes.

Deploy:
- Deploy to Vercel (recommended) by importing this repository. After deployment you can add your custom domain (ehikafashion.com) from Vercel's dashboard and follow their DNS instructions.
